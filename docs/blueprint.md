# **App Name**: Fortress Blog

## Core Features:

- User Authentication & Authorization: Enable secure user registration, login with JWT token issuance, and protected route access, with bcrypt hashing for passwords.
- AI-Powered Blog Summarization: Leverage a generative AI tool to automatically create concise summaries for blog posts upon creation or update, enhancing content discoverability.
- Blog Content Management: Authenticated users can create, edit, publish/unpublish, and delete their own blog posts. The system will handle unique slug generation.
- Public Blog Viewing: Provide a public interface for reading individual published blog posts by slug, and a paginated feed of all published content.
- Like System for Engagement: Allow authenticated users to express appreciation for blog posts through a liking mechanism, preventing duplicate likes per user.
- Interactive Commenting System: Enable authenticated users to post comments on blog entries and view all comments for a post, including author information and sorted by recency.

## Style Guidelines:

- Primary color: A rich, dependable deep blue (#245C8C), reflecting professionalism and security for key interactive elements.
- Background color: A serene, heavily desaturated blue-gray (#ECF1F5), providing a clean and approachable canvas for content.
- Accent color: A vibrant, clear cyan (#4DDADD) to highlight call-to-action buttons, interactive feedback, and important notifications.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly tech-infused and bold presence. Body font: 'Inter' (sans-serif) for exceptional readability and a clean, objective aesthetic.
- Utilize a consistent set of minimalist, outlined or solid icons to maintain clarity and enhance the professional visual appeal of the platform.
- Employ a responsive, grid-based layout with generous whitespace to ensure readability and a clean, organized presentation across devices.
- Implement subtle, performance-conscious animations for loading states, UI transitions, and interactive feedback, improving the user experience without distraction.
